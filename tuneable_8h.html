Detailed Description

Functions for specifying parameters that can be tuned by the user.

For each tuneable parameter, a corresponding slider or checkbox is created in the interface to allow the user to modify the value of that parameter. Currently, when the user changes the value of a tuneable parameter, the ESP system re-runs the entire setup() function with the tuneable parameters set to their new values.
